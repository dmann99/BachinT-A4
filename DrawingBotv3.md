When using 9x12 paper 

To get a uniform 25mm border:
* Set paper to A4
* Uncheck "Use Original Sizing"
* Set Padding to 25/17/25/25
* Cropping Mode - set as needed to show Image as desired
* Clipping mode - Drawing (?)

When loading Paper
* X : Align left side of paper with left side of Platen
* Y : Align a few MM above Y0, can butt up against bottom of control hardware screws
